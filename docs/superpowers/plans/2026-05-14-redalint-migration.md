# REDALINT WordPress → Astro Migration Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Migrate redalint.org from WordPress to a static Astro site on GitHub Pages, preserving content, SEO and old URLs, with a visual identity faithful to the current redalint.org.

**Architecture:** Build on the existing scaffold (Astro 6.3 + Tailwind 4 + DaisyUI 5 + Pagefind). Convert WP XML to MDX with the existing converter, add team/partners components, generate redirect stubs for old URLs via a prebuild script, polish with breadcrumbs/JSON-LD/edit-on-GitHub, then deploy.

**Tech Stack:** Astro 6.3 (static) · MDX · Tailwind 4 · DaisyUI 5 · Pagefind 1.5 · TypeScript strict · Node 20+ · GitHub Actions → GitHub Pages.

**Spec:** `docs/superpowers/specs/2026-05-14-redalint-migration-design.md`

**Verification model:** This is a static content site, not a library — there's no Jest/Vitest harness in the repo. Each task verifies with one or more of:
- `npm run typecheck` (Astro + TS strict)
- `npm run build` (full Pagefind-indexed production build)
- `npm run preview` + manual page visit at `http://localhost:4321/site-redalint/...`
- `grep`/`find` against `dist/` to confirm generated artifacts

If a problem only shows in the browser, open the printed URL and check the specific selector mentioned in the task.

import { getCollection, render } from 'astro:content';
import BaseLayout from '~/layouts/BaseLayout.astro';
import MembroCard from '~/components/MembroCard.astro';
import { LANGUAGES, t, type Lang } from '~/i18n/translations';

export function getStaticPaths() {

const lang = Astro.params.lang as Lang;

const allMembros = await getCollection('equipe');
const membros = allMembros

  .sort((a, b) => (a.data.order ?? 99) - (b.data.order ?? 99));

const rendered = await Promise.all(
  membros.map(async (m) => ({ data: m.data, slug: m.id, ...(await render(m)) })),

    <div class="section-divider mt-3"></div>

    {membros.length === 0 ? (
      <p class="opacity-70 mt-10">{t(lang, 'team.empty')}</p>

      <div class="mt-12">

          <MembroCard
            lang={lang}
            name={data.name}
            role={data.role}
            affiliation={data.affiliation}
            photo={data.photo}
            links={data.links}
          >

</BaseLayout>

import { getCollection, render } from 'astro:content';
import BaseLayout from '~/layouts/BaseLayout.astro';
import MembroCard from '~/components/MembroCard.astro';
import { LANGUAGES, t, type Lang } from '~/i18n/translations';
import { localizedPath } from '~/utils/paths';

export async function getStaticPaths() {
  const entries = await getCollection('equipe');
  // emit each member under every LANGUAGES value to keep URLs consistent;
  // bio language stays as authored (mostly pt).
  return entries.flatMap((entry) =>
    LANGUAGES.map((lang) => ({
      params: { lang, slug: entry.id },
      props: { entry },
    })),
  );
}

const { entry } = Astro.props;
const lang = Astro.params.lang as Lang;
const { Content } = await render(entry);
---

<BaseLayout
  lang={lang}
  title={entry.data.name}
  description={entry.data.role}
  image={entry.data.photo}
  pathInLang={`equipe/${entry.id}`}
>
  <section class="container mx-auto px-4 py-12 max-w-4xl">
    <p class="opacity-70 text-sm">
      <a href={localizedPath(lang, 'equipe')} class="link link-hover">← {t(lang, 'nav.equipe')}</a>
    </p>
    <div class="mt-6">
      <MembroCard
        lang={lang}
        name={entry.data.name}
        role={entry.data.role}
        affiliation={entry.data.affiliation}
        photo={entry.data.photo}
        links={entry.data.links}
      >
        <Content />

  </section>
</BaseLayout>

### Task C2: Prebuild script that emits redirect stubs

**Files:**
- Create: `scripts/build-redirects.mjs`
- Modify: `package.json`
- Modify: `.gitignore`

- [ ] **Step 1: Write the script**

```javascript
#!/usr/bin/env node
/**
 * Emite stubs HTML em public/<old-slug>/index.html que redirecionam para
 * a URL canônica nova. São gerados em build time (pre-astro-build) para
 * preservar URLs do WordPress antigo.
 *
 * Fonte de verdade: o array REDIRECTS abaixo (espelha docs/url-mapping.md).
 */
import fs from 'node:fs/promises';
import path from 'node:path';
import { fileURLToPath } from 'node:url';

const ROOT = path.resolve(path.dirname(fileURLToPath(import.meta.url)), '..');
const PUBLIC = path.join(ROOT, 'public');

// Base é injetado pela config do Astro. Em prod usamos o mesmo valor
// declarado em astro.config.mjs.
const BASE = process.env.PUBLIC_BASE_URL ?? '/site-redalint';

/** [oldPath, newPathInBase] */
const REDIRECTS = [
  ['institucional', 'pt/institucional'],
  ['pesquisas', 'pt/pesquisas'],
  ['publicacoes', 'pt/publicacoes'],
  ['equipe', 'pt/equipe'],
  ['equipe-acervo', 'pt/equipe-acervo'],
  ['noticias', 'pt/noticias'],
  ['como-pesquisar', 'pt/como-pesquisar'],
  ['como-colaborar', 'pt/como-colaborar'],
  ['como-colaborar-espanol', 'es/como-colaborar'],
  ['sobre-o-acervo', 'pt/sobre-o-acervo'],
  ['julieta-abba', 'pt/equipe/julieta-abba'],
  ['lancamento-da-revista-redalint', 'pt/noticias/lancamento-da-revista-redalint'],
  [
    'texto-001-desafios-da-internacionalizacao-da-educacao',
    'pt/noticias/texto-001-desafios-da-internacionalizacao-da-educacao',
  ],
];

function stubHtml(targetUrl) {
  return `<!doctype html>
<html lang="pt-BR">
<head>
<meta charset="utf-8">
<title>Redirecionando…</title>
<link rel="canonical" href="${targetUrl}">
<meta name="robots" content="noindex,follow">
<meta http-equiv="refresh" content="0;url=${targetUrl}">
<script>window.location.replace(${JSON.stringify(targetUrl)});</script>
</head>
<body>
<p>Esta página foi movida. Se você não for redirecionado automaticamente,
<a href="${targetUrl}">clique aqui</a>.</p>
</body>
</html>
`;
}

async function main() {
  let written = 0;
  for (const [oldPath, newPathInBase] of REDIRECTS) {
    const targetUrl = `${BASE.replace(/\/+$/, '')}/${newPathInBase}/`.replace(/\/+/g, '/');
    const dir = path.join(PUBLIC, oldPath);
    await fs.mkdir(dir, { recursive: true });
    await fs.writeFile(path.join(dir, 'index.html'), stubHtml(targetUrl));
    written++;
  }
  console.log(`> build-redirects: emitiu ${written} stubs em public/`);
}

main().catch((err) => {
  console.error(err);
  process.exit(1);
});
```
