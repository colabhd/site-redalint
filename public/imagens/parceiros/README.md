# Logos de parceiros institucionais

Logos de instituições e agências de fomento exibidas na faixa "Apoio
institucional" da home (e potencialmente no rodapé). Formato preferido:
SVG monocromático ou PNG transparente.

Logos esperados (use o nome de arquivo exato):

- `capes.svg` — CAPES
- `cnpq.svg` — CNPq
- `fapesp.svg` — FAPESP
- `fapergs.svg` — FAPERGS
- `clacso.svg` — CLACSO
- `unesp.svg` — UNESP
- `unco.svg` — Universidad Nacional del Comahue

Enquanto os SVGs não estão disponíveis, o componente `PartnersStrip`
exibe o nome do parceiro em texto no lugar do logo (fallback via
`onerror`).
